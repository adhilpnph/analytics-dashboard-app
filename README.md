# React Analytics Dashboard

A responsive analytics dashboard built with React.  
Includes a collapsible sidebar, route-based navigation, dark mode, animated UI components, and responsive layout behavior.

## Features

- Responsive layout (desktop and mobile)
- Collapsible sidebar (icon-only mode on collapse)
- Mobile sidebar overlay with toggle
- Sticky top navigation bar
- Dark mode toggle
- Route-based navigation using React Router
- Active link highlighting with NavLink
- Animated stat cards on load
- Chart section
- Responsive data table with horizontal scroll
- Smooth UI transitions and micro-interactions

## Tech Stack

- React
- Vite
- React Router DOM
- CSS (custom styling)
- React Switch
